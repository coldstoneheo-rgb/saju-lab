# Report Tone Guide (Draft)

## 목표 톤
- 비신비적, 설명형
- 사용자가 이해할 수 있는 언어
- 결정지원(커리어/재무) 중심이되 단정 피함
- 불확실성/추정임을 명확히 표시

## 기본 문장 패턴(샘플)
- “이 부분은 **확정된 정보**보다 **해석 비중**이 큽니다.”
- “현재 리포트는 **참고용**으로, 단정적 결론을 내리지 않습니다.”
- “출생 시간이 없어 **신뢰도는 낮음**으로 표시됩니다.”
- “커리어/재무는 **전망과 준비 방향**을 제시하는 수준입니다.”

## 권장 표현
- “가능성이 있습니다 / 경향이 있습니다 / 고려할 수 있습니다”
- “요인”, “경향”, “지표”

## 피해야 할 표현
- 확정적 예언형 문장(“무조건 성공한다”)
- 의학/재무/법률적 확정형 문장

## 섹션별 톤 힌트
### Overview
- 요약 + 정보성 안내 + Disclaimer 포함

### Career & Finance
- 선택과 리스크 관리 조언 중심
- “확률/경향” 강조

### Transparency
- 확정/추정 분리 명시
- 누락 데이터(출생시간) 영향 설명
