# Golden Fixtures (Draft)

## 목적
- 절기 기준 사주 계산의 정확성을 검증하기 위한 기준 데이터셋

## 선정 기준
- 공개적으로 검증 가능한 기준표/천문 데이터 기반
- 다양한 절기 경계 날짜 포함
- 출생시간 유/무 케이스 포함

## 후보 예시(검증 필요)
- 1990-01-01 10:30 Asia/Seoul
- 2000-02-04 12:00 Asia/Seoul (입춘 경계)
- 2010-06-21 23:59 Asia/Seoul (하지 근접)
- 2015-12-22 00:30 Asia/Seoul (동지 근접)
- 2024-02-04 08:00 Asia/Seoul (입춘 경계)

## 확정된 검증 소스
- 한국천문연구원(KASI) 24절기/태양황경 데이터: https://astro.kasi.re.kr/ (생활천문관 > 절기)
- 공개 절기표(출처 명시): https://astro.kasi.re.kr/life (생활천문관 절기표)
- 신뢰 가능한 사주 계산기(교차 검증): https://astro.kasi.re.kr/ (외부 계산기 링크는 별도 기록)

## 검증 절차
- 표준 절기표 및 외부 계산기 비교
- 일주/월주/연주 확인
- time 없는 입력에 대한 모드 확인

## 기록 템플릿(샘플)
- 입력: YYYY-MM-DD HH:mm TZ
- 기준 절기 데이터 출처:
- 계산 결과(연/월/일/시주):
- 외부 검증 결과:
- 차이 발생 시 원인 추적:

## 확정 기록(초안)

> 참고: 현재 외부 데이터 접근 제한으로 실제 절기 시각/사주 결과 검증은 보류됨.
