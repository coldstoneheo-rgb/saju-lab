# MVP UI Wireframe (Textual)

## 1) 입력 화면
- 제목: "사주 리포트 생성"
- 입력 필드
  - 생년월일(양력)
  - 출생시간(선택)
  - 성별
- 옵션
  - 시간 미상 모드 설명
- CTA 버튼: "리포트 보기"

## 2) 리포트 화면(Report-first)

### 상단 요약
- 보고서 생성 시각
- 신뢰도 배지(높음/중간/낮음)
- Disclaimer 배너

### 섹션 순서
1. Overview
2. Personality (Strengths / Blind Spots)
3. Career Trend
4. Money Trend
5. Yearly Outlook (2026)
6. Monthly Highlights
7. Action Suggestions
8. Transparency Notes

## 3) 월간/일간 간단 화면(무료)
- 이번 달/오늘의 키워드 3개
- 리포트 요약 링크
