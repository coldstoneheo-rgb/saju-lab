# MVP UI Wireframe (Textual)

## 1) 입력 화면
- 제목: "사주 리포트 생성"
- 입력 필드
  - 생년월일(양력) + 양력 기준 도움말
  - 출생시간(선택) + 시간 미상 상태 도움말
  - 성별
  - 타임존 안내: Asia/Seoul 고정
- 옵션
  - 시간 미상 모드 설명
- CTA 버튼: "리포트 보기"
- 오류 상태
  - 입력값 아래에 수정 방법 중심의 오류 문구 표시

## 2) 리포트 화면(Report-first)

### 상단 요약
- 보고서 생성 시각
- 신뢰도 배지(높음/중간/낮음)
- Disclaimer 배너
- 출생시간 미상 안내 배너

### 섹션 바로가기
- 요약
- 커리어
- 재무
- 월간
- 투명성

### 섹션 순서
1. Overview
2. Personality (Strengths / Blind Spots)
3. Career Trend
4. Money Trend
5. Yearly Outlook (2026)
6. Monthly Highlights
7. Action Suggestions
8. Transparency Notes

## 3) 월간/일간 간단 화면(무료)
- 이번 달/오늘의 키워드 3개
- 리포트 요약 링크
