# Information Architecture (MVP Draft)

## 1) 페이지 구조

1. 홈
2. 입력
3. 리포트
4. 월간/일간 요약(무료)

## 2) 사용자 흐름

- 홈 → 입력 → 리포트 생성 → 섹션 탐색
- 리포트 상단에서 월간/일간 요약으로 이동 가능
- 출생시간 미상 시 신뢰도 배지 표시

## 3) 핵심 UI 요소

### 입력
- 생년월일(양력)
- 출생시간(선택)
- 성별
- 시간 미상 안내

### 리포트
- 상단 요약 + 신뢰도 배지
- 섹션별 카드 레이아웃
- Transparency 섹션에서 추정/확정 표시

## 4) MVP 정보 우선순위
- Report-first 콘텐츠 노출
- 결정지원(커리어/재무) 섹션은 상단 배치
- Disclaimer는 상단과 하단에 모두 노출
